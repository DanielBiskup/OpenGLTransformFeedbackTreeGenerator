
\chapter{Einleitende Hinweise} \label{Einleitende Hinweise}

Eine Anmerkung zu Beginn: Das Programm kompiliert und linkt zwar unter Windows, stürzt beim Ausführen allerdings ab. Damit sie sich vorstellen können wie das Programm aussehen würde wenn sie es ausführen könnten, habe ich ihnen ein Video davon aufgenommen, wie das Programm aussieht, wenn es unter Linux ausgeführt wird. Die Videodatei heißt \code{Video.mp4}.

Dieses Projekt enthält die beiden Archive \code{Projekt\_Windows} und \code{Projekt\_Linux}. Bitte berücksichtigen sie, dass die eigentliche Abgabe die sich nach Möglichkeit bewerten sollten \code{Projekt\_Linux} ist. Dieses enthält Teilweise neuere Codeabschnitte. Der Grund dafür, dass ich zwei Projekte habe ist, dass ich das Linux Projekt auf einen Windows-Rechner kopiert habe um es dort an Windows an zu passen. Als ich es zurück auf mein Linux System kopierte waren die Zugriffsrechte auf einige Dateien wie Shared Libraries verändert und daher lief auf meine Linux Build nicht mehr. Da meine Windowsversion auch auf Windows nicht richtig funktioniert, habe ich mich entschieden, die Änderungen für Windows nicht zurück in mein Hauptprojekt \code{Projekt\_Linux} zu integrieren. Das beiliegende Linuxprojekt sollte allerdings fehlerfrei funktionieren.
